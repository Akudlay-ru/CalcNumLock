# NumLockCalc 2026 Free Core

**Версия: 10.2.0**

NumLockCalc 2026 Free Core — быстрый локальный калькулятор для Windows: NumLock-hotkey, встроенный стандартный калькулятор, единицы измерения, копирование результата, история вычислений и базовые настройки.

Публичная версия не содержит Pro-модулей, лицензирования, автозамены, дневника, скриншотов, трекера активности, управления окнами в трей, генераторов лицензий и приватных файлов. Да, иногда отсутствие функций — это не баг, а санитарная обработка перед публикацией.

## Состав публичного релиза

```text
calc_numlock_tray.pyw
functions.py
styles.py
standard_calc.py
standard_calc_engine.py
settings.json
Values.txt
build_exe.py
NumLockCalc_2026_10.exe
```

## Что умеет Free Core

- запуск и скрытие калькулятора по NumLock;
- встроенный калькулятор в стиле Windows Calculator;
- математический приоритет операций, скобки, проценты и процентные сценарии;
- копирование результата в буфер обмена;
- меню единиц измерения и быстрых вставок;
- настройка прозрачности, позиции, темы и поведения окна;
- локальное хранение настроек рядом с программой.
